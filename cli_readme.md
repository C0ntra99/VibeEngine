# Usage
<!-- usage -->
```sh-session
$ npm install -g VibeEngine_cli
$ VibeEngine COMMAND
running command...
$ VibeEngine (-v|--version|version)
VibeEngine_cli/0.0.0 linux-x64 node-v13.3.0
$ VibeEngine --help [COMMAND]
USAGE
  $ VibeEngine COMMAND
...
```
